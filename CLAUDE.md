# Cấu hình Claude Code

## Ngôn ngữ
- Sử dụng 100% tiếng Việt trong tất cả phản hồi
- Giải thích code bằng tiếng Việt
- Comment code bằng tiếng Việt

## Quy tắc làm việc
- Luôn sử dụng TodoWrite để theo dõi tiến trình
- Giải thích ngắn gọn và súc tích
- Ưu tiên chỉnh sửa file có sẵn thay vì tạo mới
- Không tự ý thay đổi tính năng, logic làm hỏng dữ liệu đã lập trình

## Nguyên tắc phát triển
- Bảo toàn logic và dữ liệu hiện có
- Chỉ cải tiến khi được yêu cầu rõ ràng
- Kiểm tra kỹ trước khi thay đổi code quan trọng
- Ưu tiên sự ổn định của ứng dụng
